DEEP LEARNING ON SATELLITE DATA

When vision meets technology

The purpose of our project is to conduct a study on the land usage composition of our areas of interest. Our project engages resources in the following sections:

1) Cloud Resource: We used Amazon Web Service (AWS) for data storage and computing resources.

2) Computing Engine: For the algorithm to learn from satellite data, we employ a deep learning model - a Convoluted Neural Network (CNN) - to be more precise, a Residual Network of 50 Activation Layers (ResNet-50). This model is recognized as the state-of-the-art image recognization tool from Microsoft to Facebook.

3) Training and Test Data: For model training and validation, we used the EuroSAT - a total of 27,000 scenes from Sentinel 2, classified into 10 categories. The data was sampled from satellite images over Europe, but feasibility test was conducted for the US, China, and Australia. This, together with the same resolution as our input data (10 meter pixel), makes it adequate for the purpose of our study. (REF: [1] Eurosat: A novel dataset and deep learning benchmark for land use and land cover classification. Patrick Helber, Benjamin Bischke, Andreas Dengel, Damian Borth. IEEE Journal of Selected Topics in Applied Earth Observations and Remote Sensing, 2019 (under review), [2] Introducing EuroSAT: A Novel Dataset and Deep Learning Benchmark for Land Use and Land Cover Classification. Patrick Helber, Benjamin Bischke, Andreas Dengel. 2018 IEEE International Geoscience and Remote Sensing Symposium, 2018 (accepted))

4) Data to Study: We use Sentinel HUB API to obtain satellite data over Houston, upon the user's request, on which we perform our machine learning study. The bands we acquired are RGB bands from Sentinel 2 L1C Full Bands Data, prioritizing the latest time, with cloud coverage under 20%.
