DEEP LEARNING ON SATELLITE DATA

When vision meets technology

The purpose of our project is to conduct a study on the land usage composition of our areas of interest. Our project engages resources in the following sections:

1) Cloud Resource: We used Amazon Web Service (AWS) for data storage and computing resources.

2) Computing Engine: For the algorithm to learn from satellite data, we employ a deep learning model - a Convoluted Neural Network (CNN) of 5 hidden layers.

3) Training and Test Data: For model training and validation, we used the Deepsat SAT-6 datasets - a total of 330,000 scenes classified into 6 categories spanning the whole of the Continental United States (CONUS). (REF: Saikat Basu, Sangram Ganguly, Supratik Mukhopadhyay, Robert Dibiano, Manohar Karki and Ramakrishna Nemani, DeepSat - A Learning framework for Satellite Imagery, ACM SIGSPATIAL 2015.)

4) Data to Study: We use planet.com API to obtain satellite data real-time, upon the user's request, on which we perform our machine learning study.
